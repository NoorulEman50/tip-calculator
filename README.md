# Frontend Assessment: Tip Calculator & Bill Splitter

A single-screen responsive tip calculator and bill splitter that calculates real-time split shares seamlessly as you type.

## How to Run Locally

Since this project is built using vanilla HTML, CSS, and JavaScript, you can run it instantly without installing heavy packages.

### Option 1: Using VS Code Live Server (Recommended)
1. Open this project folder in **Visual Studio Code**.
2. Install the **Live Server** extension if you haven't already.
3. Right-click on `index.html` and select **Open with Live Server**.
4. The application will launch automatically in your default browser at `http://127.0.0.1:5500/`.

### Option 2: Direct Browser Launch
Simply double-click the `index.html` file inside the project folder to open it directly in any modern web browser.
